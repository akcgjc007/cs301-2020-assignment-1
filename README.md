# Assignment1: Shell commands in C
This project involves making a C shell which supports following commands:
 - `cat`
 - `cd`
 - `cp`
 - `grep`
 - `ls`
 - `mkdir`
 - `pwd`

## How to run?
 - `make compile` 
 - `make run`
 - `make clean`
